# 🏥 Health Minister Application - Frontend

This repository contains the official Flutter application for the Health Minister project. It is designed with the official Ministry of Health UI guidelines.

## ✨ Key Functionalities
* **Official Branding:** Custom headers with the Ministry logo and consistent brand colors.
* **Guest Access:** Dedicated Guest Page with full clinic search and information.
* **Clinic Management:** * Search and filter clinics by location/service.
    * Detailed clinic view with directions and contact info.
* **Interactive UI:** Custom "Blue Card" components and expandable sections.
* **Theme Support:** Full Dark Mode and Light Mode support across all core screens.
* **User Flow:** Complete journey from Landing Page to Sign-in/Sign-up.

## 🛠 Project Structure
* `lib/screens/`: Contains all restored UI screens (Home, Clinic, Profile, etc.)
* `lib/widgets/`: Reusable components (Blue Cards, Ministry Headers).
* `lib/theme/`: Official color palettes and typography.

## 🚀 How to Run
1. Ensure you are on the `main` branch.
2. Run `flutter pub get` to install dependencies.
3. Run `flutter run` on your preferred device.
